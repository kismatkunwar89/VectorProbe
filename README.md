# Network Enumeration Tool

## Overview

The Network Enumeration Tool is a Python-based application designed to automate the discovery and documentation of network hosts, services, and vulnerabilities. This tool integrates various reconnaissance and enumeration techniques to provide a comprehensive analysis of network environments.

## Features

- Host discovery and availability verification
- Operating system detection
- Service enumeration
- Network topology mapping
- Comprehensive report generation in Markdown format

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/network-enumeration-tool.git
   cd network-enumeration-tool
   ```

2. Create a virtual environment:
   ```
   python3 -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage

To run the tool, execute the following command:

```
python src/main.py --help
```

This will display the available command-line options and usage examples.
